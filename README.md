Finance Dashboard

A modern, responsive Finance Dashboard built with React, Vite, and Tailwind CSS to help users track income, expenses, and financial insights with clean UI and interactive visualizations.

 Features
 Dashboard Overview
View total balance, income, and expenses at a glance
Recent transactions summary
 Transaction Management
Add, view, and manage transactions
Categorized income & expense tracking
 Insights & Analytics
Interactive charts for financial analysis
Spending trends and breakdown
 Balance Trends
Visualize balance changes over time
 Income vs Expense
Compare financial inflow and outflow
 Spending Breakdown
Category-wise expense distribution
 Responsive Design
Fully optimized for mobile, tablet, and desktop
